| Concept | Formula | Rule Name |
| :--- | :--- | :--- |

| **Product Rule** | $(uv)' = u'v + uv'$ | Differentiation |

### 💻 Technical "Github Hacks
*Quick reference for the tools and syntax used in this project.*

- **HTML Integration:** Used `<img src="...">` and `<div align="center">` tags to bypass standard Markdown layout limits, allowing for centered images and custom-sized banners.
- **Python Scripting:** Inserted `.py` files to handle backend logic. Used the Python CLI (Command Line Interface) to create a dynamic "Points Calculator" that processes user input.
- **Dynamic Badges:** Inserted API links from `Shields.io` to create colorful metadata badges. Format: `![Label](https://img.shields.io/badge/Text-Color)`.
- **Markdown Tables:** Organized complex data using the pipe `|` and hyphen `-` syntax. This allowed for structured "Information Architecture" without needing CSS.
- **Relative Linking:** Used `./filename.md` to connect separate files. This creates a "File System" that works across different environments (GitHub web vs. Local Desktop).
- **Giphy & Media:** Inserted external GIF links into Markdown to enhance the User Interface (UI) and create a "Cozy/Aesthetic" project vibe.
